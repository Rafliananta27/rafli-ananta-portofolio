01
Data Logger
Work Project Sistem Monitoring Mesin Produksi · PT Astra Daihatsu Motor · Jan–Jun 2024
+
Mengapa
Di lantai produksi ADM, data parameter mesin dicatat manual oleh operator. Proses ini rawan human error dan lambat — ketika ada anomali, tim baru mengetahuinya terlambat.
Bagaimana
Berkoordinasi dengan tim engineer untuk memahami alur data. Merancang sistem web monitoring dengan PHP (CodeIgniter 3) dan MySQL. Tantangan: belajar PLC dari nol — teknologi di luar bidang saya.
Hasil
Data parameter mesin kini terekam otomatis dan real-time. Mengurangi human error dan mempercepat deteksi anomali di lantai produksi manufaktur.
💡 Refleksi Jujur
Tantangan
Teknologi PLC sepenuhnya baru — belajar dari nol menghabiskan waktu untuk sesuatu yang akhirnya tidak jadi digunakan. Server deployment juga belum tersedia.
Yang Akan Diubah
Lebih awal mengakui keterbatasan dan meminta bantuan teknisi ahli. Belajar sendiri dari nol untuk teknologi di luar core skill membuang waktu yang bisa lebih berdampak.
Momen Berkesan
Ketika sistem berjalan di lantai produksi dan data mesin masuk real-time untuk pertama kalinya — saat itulah skill terasa nyata dan bermakna.

03
Maintenance System
Class Assignment Group Project Sistem Manajemen Pemeliharaan Korektif · Nov 2022–Jan 2023
+
Mengapa
Mesin lab kampus diperbaiki secara reaktif — rusak dulu baru ditangani. Pendekatan ini mahal dan tidak efisien. Kami ingin sistem perawatan proaktif berbasis data.
Bagaimana
Dua arsitektur web sekaligus: PHP CI3 dan ASP.NET MVC dengan SQL Server, terintegrasi langsung ke mesin fisik lab. Notifikasi otomatis ke WhatsApp dan email. Peran saya: analisis sistem, DB design, logika penjadwalan.
Hasil
Jadwal maintenance bisa dibuat, dipantau, dan dilaporkan dalam satu platform. Mendapat apresiasi langsung dari teknisi lab mesin kampus.
💡 Refleksi Jujur
Tantangan
Integrasi ke mesin fisik dan memicu notifikasi WhatsApp + email real-time. Dua arsitektur web sekaligus membuat koordinasi tim lebih kompleks dari bayangan awal.
Yang Akan Diubah
Libatkan teknisi mesin sejak awal desain, bukan hanya saat testing. Mereka yang paling tahu kebutuhan operasional nyata.
Momen Berkesan
Ketika teknisi lab berkata sistem kami benar-benar membantu pekerjaan mereka — bukan karena dapat nilai bagus, tapi karena orang yang memakai sistem merasakan manfaatnya.
